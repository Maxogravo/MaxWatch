# MaxWatch
A smartwatch I made with ESP32

Link to ESP32 module used: https://www.waveshare.com/esp32-s3-touch-lcd-1.28.htm

# Functions:
- Tells the time
- Connects to wifi
- Day and Date
- Year

# How To Use:
1. Edit the code and enter your Wifi ssid and password.
2. Flash to ESP32 with libraries provided
3. Enjoy!
